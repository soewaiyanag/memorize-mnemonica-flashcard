# MN3M0N1CA Stack Flashcard App

A React-based flashcard application for memorizing the Mnemonica stack - a popular playing card memorization system used by magicians and mentalists.

## Features

- **Two Practice Modes**
  - **Number â†’ Card**: See the position number, flip to reveal the card
  - **Card â†’ Number**: See the card, flip to reveal its position

- **Flexible Learning**
  - Practice in groups of 13 cards (52 cards total divided into 4 groups)
  - Select multiple groups simultaneously
  - Shuffle mode for randomized practice

- **Beautiful UI**
  - Smooth 3D card flip animations
  - Two theme options: Coffee and Sea
  - Responsive design with clean, flat aesthetics
  - Progress tracking

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- GitHub Pages

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Deploy

```bash
npm run deploy
```
